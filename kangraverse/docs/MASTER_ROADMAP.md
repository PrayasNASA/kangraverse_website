# Success Criteria

The project is considered complete when:

- Users can explore the map.
- Heritage sites load from PostgreSQL/PostGIS.
- Search and filters work.
- Community stories are available.
- AI assistant answers questions using project data.
- Admin can manage content.
- Website is deployed and responsive.

---

# Notes

Always complete one feature package before starting another.

Never redesign completed modules without a strong reason.

Every sprint must end with a working application.
